```bash
git clone https://github.com/julioformiga/dove.git
cd dove
./dist/dove [user]
```

## 🐍 For careful (and friendly Python) users

Requirements:

- Python >=3.8
- Rich

```bash
pip install rich
python dove.py
```

- Pyinstaller (optional)

```bash
pyinstaller -F dove.py
```
